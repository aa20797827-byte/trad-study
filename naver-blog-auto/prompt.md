# 네이버 블로그 자동화 — 블로거 탐색 + 서로이웃 + 댓글 한번에

Playwright MCP를 사용해서 아래 순서대로 전부 실행해줘.

---

## [STEP 0] 설정 읽기

`naver-blog-auto/config.json` 파일을 읽어서 아래 값들을 파악해:
- 네이버 아이디 / 비밀번호
- blogId, dailyLimit, delayMinutes
- targetNiches (탐색할 주제 목록)
- myBlogInfo.description (내 블로그 소개)

---

## [STEP 1] 네이버 로그인

1. https://nid.naver.com/nidlogin.login 접속
2. 아이디, 비밀번호 입력 후 로그인
3. 로그인 성공 확인
   - 캡차 등장 시 → 즉시 멈추고 "캡차가 떴습니다. 수동으로 해결해주세요" 알려줄 것
   - 2단계 인증 등장 시 → 즉시 멈추고 알려줄 것

---

## [STEP 2] 활발한 블로거 자동 탐색

targetNiches의 키워드들을 순서대로 사용해서 활발한 블로거를 찾아.

각 키워드마다:
1. https://section.blog.naver.com/BlogHome.naver 접속
2. 검색창에 키워드 입력 (예: "리빙", "인테리어" 등)
3. 검색 결과에서 블로그 포스팅 목록 확인
4. **최근 7일 이내에 포스팅**한 블로그만 선택
5. 각 블로그 URL 수집 (blog.naver.com/아이디 형식)

조건:
- 내 블로그(blogId)는 제외
- 이미 targets.txt에 있는 URL은 제외
- 최소 포스팅 수 10개 이상인 블로그만
- dailyLimit 수만큼 수집되면 탐색 종료

수집된 URL을 `naver-blog-auto/targets.txt`에 자동으로 저장해줘.

---

## [STEP 3] 서로이웃 신청 + 댓글 (각 블로그마다)

targets.txt의 각 블로그 URL에 대해 순서대로 처리:

### [3-1] 블로그 분석
- 해당 블로그 접속
- 최근 포스팅 3개 제목 + 내용 일부 읽기
- 블로그 주요 주제 파악

### [3-2] 서로이웃 신청
- 프로필 영역의 "이웃추가" 또는 "서로이웃 신청" 버튼 클릭
- 읽은 블로그 내용 기반으로 맞춤 메시지 생성:
  **작성 규칙:**
  - 반드시 해당 블로그 특징(주제, 글 스타일, 특이사항) 1가지 이상 구체적으로 언급
  - "안녕하세요"로 시작
  - 2~3문장
  - 자연스러운 존댓말
  - "좋은 글 감사합니다" 같은 판에 박힌 표현 절대 금지
  - 내 블로그(리빙 제품 리뷰)와 공통점 언급하면 더 좋음
- 메시지 입력 후 신청 완료

### [3-3] 댓글 등록 (commentEnabled: true인 경우)
- 가장 최근 포스팅으로 이동
- 포스팅 내용의 구체적인 부분을 언급하는 댓글 생성:
  **작성 규칙:**
  - 포스팅 내용에서 인상적인 부분 1가지 구체적으로 언급
  - 1~2문장
  - 자연스러운 반응 (공감, 질문, 추가 의견 등)
  - 광고성 표현 금지
- 댓글창에 입력 후 등록

### [3-4] 대기
- 처리 완료 후 delayMinutes분 대기
- 처리 결과 기록

---

## [STEP 4] 완료 보고

처리가 끝나면 아래 형식으로 보고해줘:

```
✅ 완료 보고
━━━━━━━━━━━━━━━━━━━━
총 처리: X명
성공: X명 / 실패: X명

[처리 목록]
1. blog.naver.com/xxx
   - 서로이웃 메시지: "..."
   - 댓글: "..."
   - 결과: 성공/실패

2. blog.naver.com/yyy
   ...
━━━━━━━━━━━━━━━━━━━━
⚠️ 오류 사항: (있으면 기록)
```

---

## 주의사항
- 오류 발생해도 다음 블로그로 계속 진행
- 이미 서로이웃인 경우 댓글만 달고 넘어가기
- 캡차 등장 시 즉시 전체 중단 후 보고
- 처리 속도는 사람처럼 자연스럽게 (버튼 클릭 후 잠깐 대기 등)
